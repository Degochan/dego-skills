# La Fragata Brand System

This reference captures the visual identity currently reflected on `lafragata.net`.

## Summary

La Fragata presents itself as a dark editorial media brand with a political-news tone. The visual language is not minimalist-tech and not corporate-blue in a generic sense. It blends:
- deep navy page backgrounds
- gold accents for hierarchy and interaction
- white to cool-light text
- glassy dark panels
- compact but premium information cards

## Official tokens

### Color palette

- `--color-bg: #011326`
- `--color-surface: #0a1f35`
- `--color-surface-alt: #111827`
- `--color-primary: #c5a059`
- `--color-primary-hover: #d4b06a`
- `--color-text: #f3f4f6`
- `--color-text-muted: #9ca3af`
- `--color-accent-live: #ef4444`

### Common supporting treatments

- Border subtle: `rgba(255, 255, 255, 0.10)`
- Border softer: `rgba(255, 255, 255, 0.05)`
- Glass panel fill: `rgba(30, 30, 30, 0.60)`
- Gold glow: `0 0 15px rgba(197, 160, 89, 0.15)`

## Typography

### Headline family

- `Montserrat`
- Preferred weights: `700`, `800`
- Use for section titles, card headings, hero titles, and strong labels

### Brand display family

- Use `Cormorant Garamond` for the main `LA FRAGATA` wordmark when the brand name is the central visual element
- The feel should be institutional, elegant, and maritime-editorial, not ornamental
- The visual feel should stay close to an engraved editorial wordmark
- This display treatment is for brand lockups, covers, seals, hero signatures, and identity-first compositions
- Do not use it for dense UI text or long paragraphs

### Body and interface family

- `Inter`
- Acceptable alternates when needed: `Roboto`, `Open Sans`
- Preferred weights: `300`, `400`, `600`
- Use for body copy, buttons, labels, metadata, and navigation

## Tone and composition

The brand should feel:
- journalistic
- firm
- contemporary
- politically aware
- premium without looking luxurious for luxury's sake

Avoid outputs that feel:
- playful
- startup-generic
- neon/cyberpunk
- pastel
- soft lifestyle magazine

## Reusable visual patterns

### Section titles

- Large, uppercase or near-uppercase
- Gold
- Tight hierarchy
- Often increased letter spacing
- If the composition is brand-led rather than article-led, the main `LA FRAGATA` lockup can switch from Montserrat to the serif brand display style

### Panels and cards

- Dark surfaces over the navy background
- Rounded corners
- Soft borders
- Subtle lift on hover
- Gold used as highlight, CTA, or selected state

### CTA treatment

- Gold button on dark background for primary CTA
- Outline or subtle glass variant for secondary CTA
- Keep labels short and direct

### Alerts and live states

- Use red only for live or urgent states
- Do not spread red across the whole interface

## Logo usage

- Default asset: `lafragatamarca/assets/logo.png`
- Alternate official direction: circular navy-and-gold emblem with ship and lighthouse
- Preserve aspect ratio
- Do not crowd the mark with nearby text
- Use on dark backgrounds whenever possible

## Modern emblem guidance

The newer logo direction supplied by the user has these traits:
- circular badge composition
- deep navy field
- gold linework and ring
- ship as the dominant symbol
- lighthouse as secondary symbol
- more premium and heraldic presence than the current web logo

Use this emblem when the piece needs:
- stronger symbolic identity
- a seal or medallion feel
- a profile image or icon-friendly mark
- a more timeless institutional look

Avoid forcing the emblem into very small horizontal header spaces unless a separate optimized asset exists.

## Practical rule

If an artifact can be mistaken for another brand after hiding the logo, it is probably not using enough La Fragata DNA. The combination of navy base, gold emphasis, Montserrat headlines, Inter UI text, and editorial card composition should remain recognizable.
