# Restore checklist

## New machine

1. Install Git.
2. Make sure your GitHub SSH key works.
3. Clone this repo into `/Users/diego/Sitios/IAcode/skills`.
4. Run `./setup/install-all.sh`.
5. Open Codex and verify the installed skills exist in `~/.codex/skills/`.

## Quick commands

```bash
ssh -T git@github.com
git clone git@github.com:Degochan/dego-skills.git /Users/diego/Sitios/IAcode/skills
cd /Users/diego/Sitios/IAcode/skills
./setup/install-all.sh
```
